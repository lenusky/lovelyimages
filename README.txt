# lovelyimages
Gestión y almacenamiento de imágenes de usuario
Aplicación que permite el alta de nuevo usuario y su posterior identificación para el almacenamiento y gestión de tus imágenes favoritas (almacenamiento,borrado y edición).
Se utiliza Html, Css y Javascript para la parte de diseño web.
Php y Sql para la gestión de base de datos.
En este caso he utilizado el servidor Wampserver y he creado las bases de datos a través del PhpMyAdmin.
En los archivos SQL, están detalladas las bases de datos que he utilizado para que se puedan crear en el caso de que no existan.
